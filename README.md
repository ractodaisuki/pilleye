# PillEye

Next.js + React + TypeScript MVP for real-time camera-assisted pill counting.

## Development

```bash
npm install
npm run dev
```

Open the app on a browser that supports `navigator.mediaDevices.getUserMedia`.
For phone camera testing, use HTTPS or a trusted local development setup.

## GitHub Pages

Pushes to `main` deploy the static export to GitHub Pages through GitHub Actions.

Expected URL:

```text
https://ractodaisuki.github.io/pilleye/
```

In the repository settings, set Pages source to GitHub Actions.
